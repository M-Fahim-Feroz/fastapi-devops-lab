# Cost Notes — FastAPI DevSecOps Pipeline

This repository runs entirely locally using Docker Compose and on GitHub Actions. There are **no cloud resources** provisioned.

## Cost Breakdown

| Resource | Cost |
|---|---|
| Docker Compose (local) | Free |
| GitHub Actions (public repo) | Free |
| Docker Hub (public image) | Free |
| **Total** | **$0** |

## Notes

- GitHub Actions provides unlimited free minutes for public repositories.
- Docker Hub allows free public image hosting.
- No AWS, Azure, or GCP resources are created by this project.
